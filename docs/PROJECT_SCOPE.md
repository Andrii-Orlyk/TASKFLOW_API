# TaskFlow API — Project Scope

## Goal

Build a Junior .NET portfolio backend that demonstrates practical web application development.

## MVP features

1. User registration
2. User login
3. JWT authentication
4. Current user endpoint
5. Projects CRUD
6. Tasks CRUD
7. Task status update
8. Task comments
9. Dashboard summary
10. Task filtering
11. Pagination
12. Validation
13. Centralized error handling
14. User data isolation
15. PostgreSQL persistence
16. EF Core migrations
17. Docker Compose
18. Unit tests
19. Integration tests
20. GitHub Actions build/test pipeline

## Not in MVP

- React frontend
- Microservices
- Kubernetes
- RabbitMQ
- Redis
- Event sourcing
- Full CQRS with MediatR
- Stripe
- AI features
- Admin panel
- Real-time notifications
- Email confirmation
- OAuth / Google login
- 2FA
- Mobile app
- Complex reports
- Multi-tenant SaaS
